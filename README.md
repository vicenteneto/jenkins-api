# jenkins-java-api

To use it in your Maven build add:
```xml
  <repositories>
    <repository>
      <id>jitpack.io</id>
      <url>https://jitpack.io</url>
    </repository>
  </repositories>
```

and the dependency:

```xml
  <dependencies>
    <dependency>
      <groupId>com.github.jitpack</groupId>
      <artifactId>maven-simple</artifactId>
      <version>0.1</version>
    </dependency>
  </dependencies>
```
